## Zug

[Zug](../design/zug.md) is a simple, fast, and secure consensus protocol used on the Casper Mainnet. Read the paper: [From Weakly-terminating Binary Agreement and Reliable Broadcast to Atomic Broadcast](https://arxiv.org/abs/2205.06314).
